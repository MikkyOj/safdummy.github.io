GROUPS OF THE DEPARTMENT

There are certain sections of the department in which all the students are to belong to one or the other. The department is divided into three major groups namely: SOFTWARE DEVELOPERS GROUP, HARDWARE GROUP AND APPLICATION PACKAGES GROUP.

  • SOFTWARE DEVELOPERS

    This is an important group of the department that has students that write codes, programs and design websites, applications and are very profficient in programming languages such as C++, C#, JAVA, Python, Go-lang and many more.

    Some of the projects this group has done include:

    1. Development of lotto base game using Java

    2. Development of Students' daily attendance for each course

    3. Development of Multi-purpose calculator using Java

    4. Design and implementation of Department website

    5. Development and implementation of Polytechnic Grading System

  • HARDWARE GROUP

    This is a sub section of the department that has students that dedicate their skills into coupling, uncoupling, troubleshooting, networking, logic and the electronic aspects of the computer system and the computer science department as a whole.

    Some of the projects this group has done include:

    1. Development of manual for uncoupling and coupling of a desktop

    2. Networking and connection of E-library

  • APPLICATION PACKAGES

    In a world world where computer systems are used on a daily basis, the Application Packages is specialized in learning and utilizing major software packages like MS Office, CorelDraw, Database mangement sytem and many more..

    Some of the projects this group has done include:

    1. Creation of School Magazine

    3. Design of Academic banners

    4. Design of departmental logo
